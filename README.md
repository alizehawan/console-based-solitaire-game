# Console-Based Solitaire Game

This project demonstrates **Data Structure concepts** in C++.

## Features
- Card deck implemented using arrays or linked lists.
- Game logic includes moving cards, checking win conditions.
- Console-based card display.
- Use of stacks/queues or arrays for managing cards.

## Concepts Highlighted
- Arrays / Linked Lists
- Stacks / Queues (if used)
- Loops and conditional statements
- Functions for modular design
